## 🚀 Instalación

1. **Verificar versión de Node.js:**
   ```bash
   node --version
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar el servidor:**
   ```bash
   npm start
   ```

4. **Para desarrollo (con auto-reload):**
   ```bash
   npm run dev
   ```

5. **Abrir en el navegador:**
   ```
   http://localhost:3000
   ```
